NOTE: This software package (ipopt-net) redistributes statically-linked Intel
oneAPI Math Kernel Library (MKL) code as part of the native binary
libipopt-3.so (Linux) and ipopt-3.dll (Windows). The redistributed MKL
components are:

  libmkl_intel_lp64  — LP64 BLAS/LAPACK interface layer
  libmkl_sequential  — Single-threaded compute layer
  libmkl_core        — Core MKL routines including Intel Pardiso

These libraries are used to provide the PardisoMkl linear solver within IPOPT.
